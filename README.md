# PROTOTYP RAPSTAR

## Beschreibung

PROTOTYP RAPSTAR ist eine deutsche Rap-Casting-Show auf YouTube, produziert von Massiv, Sido und PA Sports. Staffel 1 abgeschlossen, Staffel 2 angekündigt. Finale mit Top 3, die einen Deal beim Label Prototyp bekommen: 100.000 Euro für den Sieger, 50.000 für Platz 2, 25.000 für Platz 3. (Quelle: Hiphop.de Artikel über Label-Gründung - https://hiphop.de/magazin/news/prototyp-massiv-sido-pa-sports-gruenden-neues-label-416871)

## Teilnehmer

Die Show startet mit über 100 Bewerbern, dann Vorrunde mit 10 Episoden (jede mit 10 Teilnehmern), dann Knock-Out (10 Gewinner), dann Top 25 Shows (Episoden 15-20, 26 Artists), dann Finale (Top 3). (Quelle: YouTube Playlist "ALLE FOLGEN" - https://www.youtube.com/playlist?list=PLXmVnQUUvwJybBPQKowRA1Lra07lA1Yc8)
