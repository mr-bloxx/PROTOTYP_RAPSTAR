# PROTOTYP RAPSTAR

## Beschreibung

PROTOTYP RAPSTAR ist die härteste Rap-Casting-Show im Netz auf YouTube. Hier treffen hungrige Newcomer auf echte Rap-Veteranen. Die Jury besteht aus Massiv, Sido, PA Sports und Ali Bumaye. Die Gewinner bekommen einen Record Deal beim Label Prototyp.

## Staffeln

- Staffel 1: Abgeschlossen, Finale mit Top 3, die einen Deal bekommen.
- Staffel 2: Angekuendigt von Massiv ("Die nächsten Jahre kriegt ihr mich nicht mehr weg").
